## Example Instructions

This is an example of how to use NeurOCAST for bias correction of STOFS-2D-Global total water level forecast guidance.

## Setting Up the Environment

After setting up a Python environment and installing NeurOCAST, use the requirements.txt file to install the additional packages required for this example:

```
pip install -r requirements.txt
```

## Code Structure

- **Data_Preparation:**

Contains scripts to collect and preprocess the data required for this example.

- **Model_training:**

Includes the script used to train and validate NeurOCAST on the prepared data.

- **Figures:**

Contains scripts for generating figures and evaluating model performance.
